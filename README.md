# React-client

> Application with React, Webpack, TypeScript and Babel


## initialise project
```shell
sudo npm install
```

## Starting application

```shell
sudo npm start
```

## Building application

```shell
sudo npm run build
```

## Type-Checking the repo

```shell
npm run type-check
```

And to run in --watch mode:

```shell
sudo npm run type-check:watch
```
